# progressive-disclosure

LLM 컨텍스트 윈도우를 효율적으로 사용하는 3단계 정보 로드 원칙.

## 주요 기능

- 3단계 정보 로드 전략
- 컨텍스트 윈도우 최적화
- 점진적 정보 공개 패턴
- 효율적인 프롬프트 설계

## 사용 방법

- 참조 대상: Skills, Agents, Prompts 작성 시
- 트리거: SKILL.md, AGENT.md 작성, 프롬프트 설계, 컨텍스트 최적화
- 용도: 프롬프트 설계 및 문서 작성 가이드

## 디렉토리 구조

```
progressive-disclosure/
├── SKILL.md

```
