# kind-senior-developer

특정 커밋이나 문서의 변경 사항을 친절한 시니어 개발자처럼 분석하고 설명합니다.

## 주요 기능

- 커밋 변경사항 친절한 분석
- 시니어 개발자의 관점에서 코드 리뷰
- 변경 이유와 의도 설명
- 학습 포인트 도출

## 사용 방법

- 호출: `/kind-senior-developer`
- 트리거: 코드 분석, 커밋 설명, 변경 사항 이해, 코드 리뷰 설명, 시니어 설명, 코드 해설
- 인자: [commit hash 또는 파일 경로]
