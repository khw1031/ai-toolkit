# skills-ref

스킬 디렉토리를 스캔하여 CLAUDE.md에 Available Skills 섹션을 XML 구조로 생성합니다.

## 주요 기능

- 스킬 디렉토리 자동 스캔
- SKILL.md 메타데이터 파싱
- XML 구조의 Available Skills 섹션 자동 생성
- CLAUDE.md 자동 업데이트

## 사용 방법

- 호출: `/skills-ref`
- 트리거: 스킬 등록, 스킬 목록 생성, CLAUDE.md 스킬 섹션, skills ref, 스킬 인덱스, available skills 작성
- 용도: 프로젝트 스킬 카탈로그 관리

## 디렉토리 구조

```
skills-ref/
├── SKILL.md
├── references/
└── scripts/
```
