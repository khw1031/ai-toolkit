# create-ai-tool

사용자 요구사항을 분석하여 Skill 또는 Agent 중 적절한 유형을 결정하고 생성을 위임합니다.

## 주요 기능

- Skill vs Agent 판단 기준 제시 (컨텍스트 격리, 도구 제한, 자동 위임 여부)
- 의사결정 흐름 가이드 (의사결정 트리로 시각화)
- 핵심 차이점 비교 (컨텍스트, 목적, 호출 방식, 도구 제어, 규칙 적용)
- 사용 사례별 라우팅 (코딩 컨벤션, 단계별 지침, 배포/DB 작업, 코드 리뷰, 병렬 분석 등)
- 유형 결정 후 create-skill 또는 create-agent로 자동 위임

## 사용 방법

- 호출: `/create-ai-tool` (또는 트리거 키워드: 도구 만들기, 스킬 vs 에이전트, 어떤 걸 만들어야 할지, 도구 유형 선택)
- 인자: 없음 (대화형 프롬프트로 진행)
